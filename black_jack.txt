class Bank (объект создается при запуске игры, первоначально банк будет равен 0 и в процессе игры изменяться)
1)	атрибуты
•	money (хранит количество денег в банке)
2)	методы
•	add_money (добавляет деньги в банк)
•	remove_money (возвращает деньги из банка)
module Points (модуль для обработки очков игрока и дилера, подключается в соответствующие классы)
1)	атрибуты
•	points (хранит количество очков)
2)	методы
•	count_points (считает количество очков)
class Card (класс для создания объектов карт, нужен для класса CardDeck, чтобы создать колоду и тд.)
1)	атрибуты 
•	card_value (достояние карты)
•	card_suit (масть карты)
•	color (цвет карты)
class CardDeck (объект создается при запуске игры)
1)	атрибуты
•	cards (возвращает карты)
2)	методы
•	create_deck (создает колоду карт)
•	remove_card (удаляет карту из колоды)
•	show_deck (возвращает колоду карт)
•	take_random (выдает случайные карты игрокам)
module Actions (модуль действий, подключается к классам Player и Dealer)
1)	атрибуты 
•	money (деньги игрока)
•	points (очки игрока)
•	cards (карты игрока)
•	name (имя игрока или дилера)

2)	методы 
•	skip_move (пропускает ход)
•	add_card (добавляет карту)
•	show_cards (вскрывает карты)
class Player (класс игрока)
class Dealer (класс дилера)
class Game (основной класс игры)
1)	методы
•	start (начинает игру)
•	result (подводит итоги игры)
•	restart (начинает игру заново)
